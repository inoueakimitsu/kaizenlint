## 通常英語表記する用語

固有名詞や略語など、通常英語で表記する用語を無理にカタカナ表記にしないでください。
ただし、日本語に定着したカタカナ外来語（サンドボックス、ホワイトリスト、データベース、サーバー、キャッシュなど）はカタカナのまま使用してください。英語に戻す必要はありません。

このルールが対象とするのは、英語のまま書くのが自然な固有名詞・略語・技術ブランド名です。

**Bad:**
```python
# エラスティック アイ ピーを設定します。
# エス キュー エルクエリを実行します。
```

**Good:**
```python
# Elastic IP を設定します。
# SQL クエリを実行します。
# サンドボックス環境で実行します。  ← カタカナ外来語はそのままでOK
# ホワイトリストに追加します。      ← カタカナ外来語はそのままでOK
```
