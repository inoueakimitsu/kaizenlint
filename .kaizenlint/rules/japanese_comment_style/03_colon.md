## コロンの日本語的使用

「です:」のようにコロンを英語のように使うのは避けて「です。」のように言い換えてください。

**Bad:**
```python
# ～は以下の通りです:
```

**Good:**
```python
# ～は以下の通りです。
```

**Bad:**
```python
# 例: "abc_class/1.0.0"
```

**Good:**
```python
# 例として "abc_class/1.0.0" のような文字列です。
```
